# rbh

A simple CLI tool to generate images suitable for RedBubble. Uses a SVG as input.

## Dependencies

ImageMagick 6 (not 7)
Inkscape
JDK 8 or later

## Installation

Build with:
```
lein bin
```

## Usage

```
$ rbh [OPTIONS] file ...
```

## Options

```
-d, --debug    Sets log level to debug
-h, --help     Print this help message
-v, --version  Print the current version number of rbh.
```
